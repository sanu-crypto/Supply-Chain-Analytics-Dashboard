Supply Chain Analytics Dashboard

This project is an interactive Power BI dashboard developed for analyzing supply chain and inventory data. The dashboard provides insights into stock quantity, order quantity, supplier performance, warehouse analysis, delivery time trends, and product distribution through interactive charts and KPI cards.

The dashboard helps in monitoring inventory performance, understanding supplier and warehouse trends, and improving supply chain decision-making using data-driven analytics.
